# Citrix-Inc-Interview
Android app that consumes Citrix' APIs to display Users, Roles their Organizations. `Citrix Interview Solution`, for the Android Developer Role

### Built with:
- Jetpack Compose
- Material-You
- Retrofit
- Kt Flows
- Room Database
- Hilt
